Position

The position property specifies the type of positioning method used for element. The four position values are:

  • fixed: this will pin something to a spot, and it will stay th as you scroll. (ex: nav bar) Fixed position takes elements out of the normal documflow.
  • static: this is the default setting, it is not affected by toottom, left, and right properties. It is always positioned according to the normalw of the page.
  • relative: does not take the element out of normal document flStill appears within its space / div /container, but allows it to be offset a bit fro e top, bottom, right, left. Top, right, bottom, and left properties will move it a from its normal position. Other content will not be adjusted to fit into any gap l by the element.
  • absolute: takes element out of the normal document flow. If aent element position is set to relative, and the child element is set to absolute, child element's new ( 0 , 0 ) coordinate will be the upper left hand corner of the pat element instead of the whole page.

The z-index property allows you to override the normal stacking r of the page. The default z-index is 0 , and you can set that value to be positive negative. An element with a greater z-index will stack in front of an element with ower z-index. If two elements have the same z-index, or don't have one assigned, t lement positioned last in the HTML will show on top. You must have a position assid in order to assign a z-index.
